# LifeTask Landing Page

Static landing page for `trylifetask.com`.

## Run Locally

From the repo root:

```bash
npx serve landing
```

Or open `landing/index.html` directly in a browser.

## Deploy On Vercel

Use these Vercel settings:

- Framework preset: `Other`
- Root directory: `landing`
- Build command: leave empty
- Output directory: `.`

Point `trylifetask.com` to the Vercel project after deployment.

## Assets

The page uses real LifeTask app preview images copied into `landing/assets`.
